O HTML DOM (Document Object Model)

O dom é o seu documento inteiro, e com o dom podemos deixar nosso HTML dinâmico através do js

Como Manipular o DOM?

Através dos Id's, Classes e Tag's dos elementos no HTML podemos criar variaveis no JS que referenciam o elemento no HTML.

Métodos

Encontrando Elementos HTML

  • Document.getElementById: A maneira mais comum de acessar um elemento HTML é usar o id do elemento.

  • document.getElementsByTagName: Uma alternativa para acessar os elementos é a Tag do elemento.

  • document.getElementsByClassName: Tembém podemos acessar elemento através da classe dele.

  • Document.querySelector: De uma forma mais geral usamos o querySelector para selecionar o elemento igual ao css, se um um ID usamos #, se for classe usamos o ponto(.) e se for uma tag usamos a tag

Exemplo: No HTML você cria um Button e atribui a ele o id '#buttonStart'. Para adicionar um efeito a esse botão você pode criar uma variavel e usar essa variavel com os metédos do DOM no Js. Atribuindo o Id a uma variavel: var btnStart = document.querySelector('#buttonStart')

Alterando Elementos HTML

  • element.innerHTML = `novo conteudo HTML`: Alterar o HTML interno de um elemento.

  • element.attribute = novo valor: Altere o valor do atributo de um elemento HTML

  • element.style.property = `novo estilo`Alterar o estilo de um elemento HTML

Exemplo: Criamos uma div de atribuimos a ela uma classe chamada 'container'. usamos o querySelector e salvamos essa diva na variavel container container no js. Para alterar o conteudo faremos da seguinte forma: container.innerHTML = `adicionamos o novo conteudo`
