Métodos array

CONCAT

Concatena 2 arrays

Exemplo:
var array1 = ['arroz', 'feijão', 'carne']
var array2 = ['refrigerante', 'suco', 'cerveja']
var concatenados = array1.concat(array2)
resultado:['arroz', 'feijão', 'carne', 'refrigerante', 'suco', 'cerveja']

PUSH

serve para adicionarmos elementos no final do array

Exemplo: var array1 = ['branco', 10, 20, 30];
array1.push('novo elemento');
console.log(array1);
resultado: ['branco', 10, 20, 30, "novo elemento"]

UNSHIFT

O “unshift()” faz o contrário do que o “push()” faz, ele adiciona um elemento no início de um array.

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
array1.unshift('novo elemento');
console.log(array1);
resultado: ["novo elemento", "branco", 10, 20, 30, "novo elemento"]

POP

O “pop()” remove o último elemento de um array:

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
array1.unshift('novo elemento');
console.log(array1);
resultado: ["novo elemento", "branco", 10, 20, 30, "novo elemento"]

SHIFT

O “shift()” é o contrário do “pop()”, ele remove o PRIMEIRO elemento do array.

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
array1.shift();;
console.log(array1);
resultado: [10, 20, 30, "novo elemento"]

JOIN

O “unshift()” faz o contrário do que o “push()” faz, ele adiciona um elemento no início de um array.

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
array1.unshift('novo elemento');
console.log(array1);
resultado: ["novo elemento", "branco", 10, 20, 30, "novo elemento"]

SLICE

O “slice()” extrai uma sessão de um array e retorna um novo array, veja:

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
var novoArray = array1.slice(1,3);
console.log(array1);
resultado: [10, 20]

SPLICE

No “splice()” podemos escolher um índice inicial e final para substituirmos valores entre eles

Exemplo: var array1 = ['branco', 10, 20, 30, "novo elemento"];
array1.reverse();
console.log(array1);
resultado: ["novo elemento", 30, 20, 10, "branco"]

REVERSE

O “join()” puxa elementos de um array e lista no formato de string

Exemplo: var array1 = ['branco', 10, 20, 30];
array1 = array1.join('-');
console.log(array1);
resultado: branco-10-20-30
